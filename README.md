**CODIUM**

Codium is an educational 2D Unity game that makes learning programming fun and interactive.
Players solve coding challenges through mini-games, earn coins and XP, and unlock new shrines as they level up.


**Game Overview**

Codium combines gameplay and coding practice. Each shrine presents a unique programming challenge:

- **Shrine One**: Pick and drop the codeblocks to fill in the blanks.

- **Shrine Two**: Catch falling code snippets while avoiding errors.

- **Shrine Three**: Print the answer and carry the codeblocks to submit.

Between shrines, players can change outfits and view their profile stats such as level, XP, and total coins.


**Features**

- Interactive Learning: Learn coding concepts through hands-on gameplay.

- Cloud Save Integration: Player progress, coins, and XP are stored with Unity Cloud Save.

- Leaderboard System: Compete with others using Unity Leaderboards.

- Player Customization: Change outfits and animations across pages.

- Rewards & Progression: Earn coins and level up through successful shrine clears.

- Dynamic UI: Smooth transitions between menu, world, and shrine scenes.
